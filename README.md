# FlowDoc Editor

一个基于 Electron 的本地所见即所得文档编辑器，交互方式接近飞书云文档和 Typora。

## 功能

- 新建文档时先选择保存位置，随后进入编辑。
- 选中文本后自动弹出悬浮格式菜单。
- 文本光标左侧会实时跟随显示块级 `+` 菜单。
- 光标所在位置可直接插入图片、附件、代码块和引用块。
- 支持图片粘贴、附件下载、代码高亮、撤销重做和缺失资源占位提示。

## 资源目录

- 开发环境下，图片和附件默认放在项目目录下的 `images` 与 `attachments`。
- 打包后的程序会在 `FlowDoc.exe` 所在目录自动创建 `images` 与 `attachments` 文件夹。
- 如果文件重名，会自动重命名，避免覆盖已有资源。

## 打包

```bash
npm install
npm run dist
```

打包完成后，产物会输出到 `release` 目录，包含：

- `FlowDoc-1.0.0-x64-portable.exe`
- `FlowDoc-1.0.0-x64-setup.exe`

其中便携版 `.exe` 可以直接双击运行并固定到任务栏；安装版适合长期安装使用。
